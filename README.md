# SlideOCR - Professional Presentation Digitizer

SlideOCR is a high-performance, privacy-focused tool designed to extract text from presentations (PDF, PPTX) and images with precision. Built for professionals who need to digitize content quickly and securely.

## 🚀 Key Features

- **Instant OCR**: Extract text from slides and images in real-time.
- **Privacy First**: All processing happens locally on your device. Your data never leaves your browser.
- **Multi-Format Export**: Save your extracted text as PDF, DOCX, TXT, or Excel.
- **Smart Search**: Quickly find specific keywords across all processed slides.
- **Local History**: Manage and revisit your previous extractions without cloud storage.

## 🛠️ Built With

- **React 19** & **TypeScript**
- **Tailwind CSS** for modern, responsive UI
- **Tesseract.js** for client-side OCR
- **Lucide React** for professional iconography
- **Motion** for smooth, fluid animations

## 📦 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/slideocr.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
